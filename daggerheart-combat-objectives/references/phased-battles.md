# Phased Battles

Use phases when the scene should evolve instead of repeating the same exchange.

## Good Phase Changes

- the battlefield changes shape or safety
- the objective changes from fight to chase or rescue
- reinforcements or a new threat arrive
- the main foe changes behavior or form
- allies become liabilities or vice versa

## Phase Design Rule

The phase should come from the fiction. Foreshadow when possible so the shift feels surprising but earned.

## If You Did Not Prep A Phase

If the PCs are winning easily, it is fine to let them win cleanly. Add a phase only if it makes the story stronger, not because every boss needs one.
