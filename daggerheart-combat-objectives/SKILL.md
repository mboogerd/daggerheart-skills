# Daggerheart Combat Objectives

Use this skill when the user has a fight but not yet a dramatic reason for it.

## Workflow

1. Read `references/core-procedure.md`.
2. Read `references/phased-battles.md` if the fight needs escalation or a mid-battle change.
3. Read `references/examples.md` for objective packages.
4. Read `references/edge-cases.md` if the fight risks becoming a flat slugfest.
5. Use `assets/template.md` when the user wants a structured objective brief.
6. Choose one enemy objective and one player-facing tension.
7. Add a failure cost that does not require a total party defeat.
8. Hand off to `daggerheart-encounter-budgeting` or `daggerheart-scene-planning` as needed.

## Output

- Core objective
- Immediate complication
- Cost of delay
- Win conditions
- Partial victory conditions
