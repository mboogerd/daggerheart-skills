# Daggerheart Scene Planning

Use this skill when the user needs one scene to do real work.

## Workflow

1. Read `references/core-procedure.md`.
2. Read `references/scene-pressure.md`.
3. Read `references/scene-types.md` if the scene needs a stronger mode of play.
4. Read `references/examples.md` for compact scene builds.
5. Read `references/edge-cases.md` if the scene risks stalling or repeating prior beats.
6. Use `assets/template.md` when the user wants a standard scene brief.
7. Define what the scene reveals, pressures, and changes.
8. Hand off to `daggerheart-combat-objectives` or `daggerheart-countdowns` if needed.

## Output

- Scene purpose
- Opening image
- Pressure
- NPC or environment leverage
- Possible turns
- Exit states
