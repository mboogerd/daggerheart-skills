# Daggerheart Countdowns

Use this skill when the user needs visible pressure that can advance during or between scenes.

## Workflow

1. Read `references/core-procedure.md`.
2. Read `references/examples.md` for modeled countdowns.
3. Read `references/edge-cases.md` if the countdown risks becoming invisible, punitive, or too abstract.
4. Use `assets/template.md` when the user wants a ready-to-fill clock format.
5. Choose the countdown type that matches the pressure.
6. Define the fictional trigger for each advance.
7. Name what happens at 0.

## Output

- Countdown type
- Starting value
- Advancement trigger
- Step-by-step escalation
- 0-state outcome
