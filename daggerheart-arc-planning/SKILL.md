# Daggerheart Arc Planning

Use this skill when the user wants a 3 to 4 session arc rather than one isolated session.

## Workflow

1. Read `references/core-procedure.md`.
2. Read `references/plot-threads.md`.
3. Read `references/campaign-foundations.md` if the arc needs stronger links to map, backstory, or frame-level pressure.
4. Read `references/examples.md` for worked arc skeletons.
5. Read `references/edge-cases.md` if the story is drifting, overloaded, or too linear.
6. Use `assets/template.md` when the user wants a reusable arc brief.
7. Build one arc around a change in pressure, not a full campaign outline.
8. Hand off to `daggerheart-session-planning` for the next concrete session.

## Output

- Arc premise
- A-plot
- B-plot
- C-plot
- 3 to 4 major beats
- How the arc can end in success, cost, or complication
