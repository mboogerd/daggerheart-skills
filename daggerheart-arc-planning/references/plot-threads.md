# Plot Threads

Use three thread levels:

- **A-plot:** the main pressure in the current arc.
- **B-plot:** the secondary complication that can become next arc's main pressure.
- **C-plot:** a low-volume seed for later revelation or escalation.

## Cadence

- Put A-plot pressure in almost every session.
- Let B-plot interrupt or complicate A-plot.
- Use C-plot sparingly so it feels intentional when it pays off.

## Escalation Pattern

- A-plots usually resolve or transform by the end of the arc.
- B-plots often become the next arc's A-plot.
- C-plots should leave breadcrumbs without stealing focus.

## Thread Sources

Good threads come from:

- unresolved backstory
- faction ambitions
- map locations the players care about
- fallout from previous victories
- threats that grow while the PCs act elsewhere

## Failure Mode To Avoid

Do not build three unrelated plots just because the labels exist. A, B, and C should feel like parts of one campaign, not three separate shows.
