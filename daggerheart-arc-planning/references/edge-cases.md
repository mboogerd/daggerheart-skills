# Edge Cases

- If the arc feels like a whole campaign, cut it down to one major pressure and one serious complication.
- If every beat is combat, add one investigative or relational turn that changes the meaning of later fights.
- If the arc ignores the PCs, pull one backstory, oath, or map location into the central pressure.
- If the arc has no room for failure, define what changes when the PCs are late, wrong, or divided.
