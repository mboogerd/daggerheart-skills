# Examples

## Example: Tier 3 Standard To Tier 1

- Keep role: Standard
- Replace core numbers with Tier 1 benchmarks
- Remove or soften any feature that deals Tier 3-level burst damage
- Keep the faction identity feature if it still fits the lower-tier fight

## Example: Tier 1 Bruiser To Tier 2

- Raise thresholds and damage first
- Add HP if the enemy should remain a front-line problem
- Consider one stronger positioning feature instead of only raising numbers
