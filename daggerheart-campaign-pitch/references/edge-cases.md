# Edge Cases

- If the premise has more than one main threat, collapse them into one visible pressure and move the rest into later arcs.
- If the pitch is all lore and no urgency, add a "what just changed?" sentence.
- If the pitch sounds generic, tie it to one striking place, institution, or wound in the world.
- If the premise excludes too many character concepts, widen the "why the PCs matter" language instead of flattening the conflict.
