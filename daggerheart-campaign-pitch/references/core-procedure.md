# Pitch Shape

A strong Daggerheart campaign pitch needs five things:

- A clear external problem.
- A clear emotional promise.
- A type of hero the campaign rewards.
- A starting location or situation.
- A reason action matters now.

## Build Order

1. Name the threat or instability.
2. Name what kind of story this is.
3. Name what kinds of characters fit naturally.
4. Name what has just changed.
5. End with a question or urgent problem.

## Good Defaults

- Keep the pitch specific enough to inspire characters.
- Keep the scope narrow enough that session zero can still shape the setting.
- If unsure, choose one strong problem rather than three weak ones.
