# Daggerheart Campaign Pitch

Use this skill when the user has a vague campaign idea and needs a playable premise.

## Use This Skill For

- Choosing a core conflict.
- Setting tone and themes.
- Defining campaign scope and complexity.
- Producing a short pitch for players.

## Workflow

1. Read `references/core-procedure.md`.
2. Read `references/examples.md` if the user needs models to imitate.
3. Read `references/edge-cases.md` if the premise is too broad, too crowded, or too lore-heavy.
4. Use `assets/template.md` when the user wants a structured output.
5. Choose one central pressure, one emotional promise, and one starting problem.
6. Write a pitch in 2 to 4 sentences.
7. Hand off to `daggerheart-session-zero` or `daggerheart-campaign-frame-selection` if the user wants to continue.

## Output

- Pitch
- Tone
- Themes
- Starting pressure
- Why the PCs matter
