# Daggerheart Encounter Budgeting

Use this skill when the user wants a rough encounter build for a party.

## Workflow

1. Read `references/core-procedure.md`.
2. Read `references/examples.md` for budgeted encounter patterns.
3. Read `references/edge-cases.md` if the scene uses unusual assumptions such as all-minion swarms or multiple solos.
4. Use `assets/template.md` for a standard calculation format.
5. Compute the starting budget.
6. Adjust for easier, harder, longer, or stranger fights.
7. Spend the budget and report the math.
8. Hand off to `daggerheart-adversary-selection` if the mix is still unclear.

## Output

- Party assumptions
- Starting points
- Adjustments
- Final spend
- Resulting encounter mix
