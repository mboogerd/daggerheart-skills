# Daggerheart Campaign Frame Selection

Use this skill when the user wants to pick or compare campaign frames.

## Workflow

1. Read `references/core-procedure.md`.
2. Read `references/frame-structure.md` if the user is choosing between close options or wants to customize a frame.
3. Read `references/examples.md` for concise recommendation patterns.
4. Read `references/edge-cases.md` if the table wants a hybrid frame or none of the defaults fit cleanly.
5. Use `assets/template.md` for comparison tables or recommendation briefs.
6. Match the table's tone, complexity tolerance, and genre appetite.
7. Present 1 to 3 best-fit frames.
8. Hand off to `daggerheart-session-zero` or `daggerheart-arc-planning` after selection.

## Output

- Recommended frame
- Why it fits
- What it asks of the GM
- What kinds of characters it supports
