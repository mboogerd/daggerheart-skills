# Daggerheart Session Zero

Use this skill when the user is launching a campaign and needs a structured conversation, not yet a plot.

## Workflow

1. Read `references/core-procedure.md`.
2. Read `references/cats.md`.
3. Read `references/worldbuilding.md` if the table wants to create setting material together.
4. Read `references/examples.md` for good prompting cadence.
5. Read `references/edge-cases.md` if time is short or the table already has strong opinions.
6. Use `assets/template.md` when the user wants a ready-to-run worksheet.
7. Produce an agenda and a concrete question list.
8. Hand off to `daggerheart-arc-planning` once the characters and priorities exist.

## Output

- Agenda
- Safety/content questions
- Worldbuilding prompts
- Character-connection prompts
- End-of-session decisions to lock in
