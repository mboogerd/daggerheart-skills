# Beats

A beat is any moment that changes the direction of the story.

## Prep Rule

Prepare beats, not scripts. A beat can survive player surprise. A script usually cannot.

## Good Beat Types

- Reveal
- Choice
- Reversal
- Escalation
- Cost
- Arrival
- Interruption

## Beat Planning Method

Prepare 3 to 5 beats that could plausibly happen if play continues in the current direction. Each beat should:

- change information, danger, or relationships
- arise naturally from NPC motives, countdowns, or player priorities
- be movable to a different scene if the players surprise you

## Opening Beat Rule

Start sessions with motion. Open on a threat, decision, aftermath, arrival, or emotionally charged quiet scene that points toward imminent action.

## Closing Beat Rule

If possible, end the session on a revelation, hard choice, consequence, or next-step hook rather than administrative wrap-up.
