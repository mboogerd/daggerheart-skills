# Session Prep Checklist

Answer these before play:

- What changes if the PCs do nothing?
- Which threat advances off-screen?
- Which NPC or faction acts next?
- What does each likely scene want from the PCs?
- What is the clearest hook into the opening beat?
- What clue, cost, or twist could end the session cleanly?

## Additional Checks

- Which character background or connection deserves pressure this session?
- Is there room for downtime, recovery, or interpersonal scenes?
- If combat happens, what is it about besides defeating enemies?
- Which prepared element can be dropped if pacing becomes slow?
