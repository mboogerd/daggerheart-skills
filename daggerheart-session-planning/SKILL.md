# Daggerheart Session Planning

Use this skill when the user wants concrete prep for the next session.

## Workflow

1. Read `references/core-procedure.md`.
2. Read `references/prep-checklist.md`.
3. Read `references/encounter-shaping.md` if the session may contain combat or a major confrontation.
4. Read `references/examples.md` for sample session briefs.
5. Read `references/edge-cases.md` if the next session must recover from drift, a cliffhanger, or unexpected player choices.
6. Use `assets/template.md` when the user wants a structured prep sheet.
7. Prepare only the next session's likely pressure points.
8. Hand off to `daggerheart-scene-planning`, `daggerheart-countdowns`, or combat skills as needed.

## Output

- Session purpose
- 3 to 5 expected beats
- Active threats
- Open questions
- Likely scenes
- Exit hooks for next session
