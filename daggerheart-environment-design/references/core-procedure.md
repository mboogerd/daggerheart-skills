# Environment Template

Every environment needs:

- Name
- Tier
- Type
- One-line description
- Impulses
- Difficulty
- Potential adversaries
- 2 to 3 features

## Feature Rule

Environment features should either change position, impose cost, escalate danger, or introduce something new into the scene.

## Scaling Rule

When scaling an environment, adjust Difficulty, damage, and summoned threats before rewriting the concept.

## What Each Field Does

- **Type:** tells you what style of scene the environment most naturally supports.
- **Description:** gives the table a quick picture.
- **Impulses:** summarize what the place or event tends to do to people in it.
- **Difficulty:** baseline target number for directly opposing the environment.
- **Potential adversaries:** the creatures or factions that naturally fit here.
- **Features:** the actual GM tools that make the environment matter in play.

## Environment Goal

An environment is not decorative scenery. It is a way to organize hazards, background forces, and scene pressure so the world itself can take part in play.
