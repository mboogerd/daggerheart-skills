# Environment Types

- **Exploration:** mystery, discovery, wonder, and hidden danger.
- **Social:** interpersonal leverage, attention, status, and consequence.
- **Traversal:** movement, terrain, exposure, and crossing danger.
- **Event:** pressure defined by what is happening rather than where it happens.

## Type Rule

Pick the type that matches the main pressure, not the scenery.

## Practical Use

- Use **Exploration** when the place itself holds answers, secrets, or awe.
- Use **Social** when the environment is full of norms, observers, reputation, or negotiation pressure.
- Use **Traversal** when movement and exposure are dangerous in themselves.
- Use **Event** when the central problem is a ritual, siege, celebration, uprising, attack, or other unfolding situation.
