# Daggerheart Environment Design

Use this skill when the scene itself should exert pressure.

## Workflow

1. Read `references/core-procedure.md`.
2. Read `references/environment-types.md`.
3. Read `references/feature-patterns.md` if the environment needs stronger scene-shaping tools.
4. Read `references/examples.md` for sample environment builds.
5. Read `references/edge-cases.md` if the environment is decorative, too punishing, or not tied to the scene.
6. Use `assets/template.md` for stat-block style output.
7. Pick the type that best matches the scene's pressure.
8. Create 2 to 3 features that change player decisions.
9. Add a Fear feature if the environment needs escalation.

## Output

- Name
- Tier
- Type
- Description
- Impulses
- Difficulty
- Potential adversaries
- 2 to 3 features
