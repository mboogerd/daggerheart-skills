# Tier 3 Standard

Use for competent, dangerous mid-high-tier line troops.

- Safe band: Difficulty 16-18; thresholds 18/30 to 22/34; HP 5-6; Stress 3-4; ATK +3 to +4; damage 3d8+3 to 3d10+4
- Default center: Difficulty 17; thresholds 20/32; HP 5; Stress 3; ATK +3; damage 3d8+4
- Feature emphasis: one faction-defining feature plus one simple tactical rider if needed
- Exemplars: Thorn Knight, Vampire Noble
