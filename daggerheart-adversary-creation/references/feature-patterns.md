# Feature Patterns

A good feature should tell the GM how this adversary changes play.

## Useful Feature Jobs

- **Positioning:** pull, push, knock back, pin, separate, or block movement.
- **Pressure:** mark Stress, remove Hope, impose disadvantage, or create vulnerability.
- **Protection:** resist damage, redirect attacks, shield allies, or escape focus.
- **Tempo:** attack multiple targets, act on a countdown, or gain extra spotlight opportunities.
- **Faction identity:** poison, chain, bramble, fire, ritual, illusion, command, and so on.

## Passive / Action / Reaction Use

- **Passive:** use when the trait should always define the enemy.
- **Action:** use when the GM should choose to spend the enemy's spotlight on that effect.
- **Reaction:** use when the enemy should punish or answer specific triggers.

## Signature Feature Rule

At least one feature should make the GM immediately understand why this enemy is not interchangeable with another stat block of the same role.
