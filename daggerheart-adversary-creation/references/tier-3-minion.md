# Tier 3 Minion

Use for dangerous but still disposable lesser creatures.

- Safe band: Difficulty 16-17; thresholds None; HP 1; Stress 1; ATK -1 to +1; fixed damage 6-8; `Minion (7)` to `Minion (9)`
- Default center: Difficulty 17; thresholds None; HP 1; Stress 1; ATK +0; damage 7; `Minion (8)`
- Feature emphasis: keep the basic minion package unless the fiction needs a tiny reactive twist
- Exemplars: Idolizing Imp, Elemental Spark, Tree Sapling
