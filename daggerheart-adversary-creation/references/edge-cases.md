# Edge Cases

- If the adversary has no clear battlefield job, pick one before writing features.
- If every feature spends Fear or Stress, the enemy may be unusable in practice.
- If the enemy can do everything, cut until the theme and role are obvious.
- If the stat block is only numbers, add one fictional hook or signature behavior.
- If the role is unclear but the prompt implies leadership, crowd control, or a centerpiece foe, ask instead of guessing.
- If the requested enemy is far outside the safe band for its role and tier, either explain the deviation or hand off to scaling guidance.
